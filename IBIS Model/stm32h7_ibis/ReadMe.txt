===============================
IBIS files for STM32H7 devices:
===============================

The package contains :


   + IBIS Files for STM32H743_753 devices on different packages :

	  o LQFP208
	  o LQFP176
	  o LQFP144
	  o LQFP100
	  o TFBGA240
	  o TFBGA100
	  o UFBGA176
	  o UFBGA169

   + IBIS Files for STM32H742 devices on different packages :

	  o LQFP208
	  o LQFP176
	  o LQFP144
	  o LQFP100
	  o TFBGA240
	  o TFBGA100
	  o UFBGA176
	  o UFBGA169

   + IBIS Files for STM32H750 devices on different packages :

	  o LQFP100
	  o TFBGA240
	  o UFBGA176

   + IBIS Files for STM32H745_755 devices on different packages :

	  o LQFP144
	  o LQFP176
	  o LQFP208
	  o TFBGA240
	  o UFBGA176

   + IBIS Files for STM32H747_757 devices on different packages :

	  o LQFP176
	  o LQFP208
	  o TFBGA240
	  o UFBGA169
	  o UFBGA176
	  
   + IBIS Files for STM32H7[A/B]7 devices on different packages :

	  o LQFP64 
	  o LQFP100
	  o LQFP144
	  o LQFP176
	  o TFBGA100
	  o TFBGA216
	  o TFBGA225
	  o UFBGA169
	  o UFBGA176+25
	  o WLCSP132
	  
   + IBIS Files for STM32H7B0 devices on different packages :

	  o LQFP64 
	  o LQFP100
	  o LQFP144
	  o LQFP176
	  o UFBGA169
	  o UFBGA176+25
	  
   + IBIS Files for STM32H725 devices on different packages :
 
	  o LQFP100
	  o LQFP144
	  o LQFP176
	  o VFQFPN68
	  o WLCSP115
	  o UFBGA169
	  o UFBGA176+25
	
   + IBIS Files for STM32H723 devices on different packages :
 
	  o LQFP100
	  o LQFP144
	  o TFBGA100
	  o UFBGA144
	  
   + IBIS Files for STM32H730 devices on different packages :
 
	  o LQFP100
	  o LQFP144
	  o LQFP176
	  o TFBGA100
	  o UFBGA144
	  o UFBGA169
	  o UFBGA176+25
